Apasih HTML itu?

HTML adalah sebuah markah yang digunakan untuk membuat sebuah halaman web, menampilkan berbagai informasi di dalam sebuah penjelajah web Internet dan pemformatan hiperteks sederhana HTML adalah sebuah standar yang digunakan secara luas untuk menampilkan halaman web.
